## 1. 0.8 版本定位与野心
打到了 0.6 和 0.7，我们已经拥有了登峰造极的“战役演算与指挥官交锋”功能。但战争绝不仅是将帅的艺术，更是**国家工业潜力的碰撞**。
如果游戏只有地图上初始放好的那些部队，那它仅是一个“解谜战棋”。加入了 **0.8 版本的初级经济与生产系统**后，游戏才真正跨入了《钢铁雄心》式的“**4X大战略（大局观探究）**”维度。
我们要实现：招募新军、前线战损后方补员、争夺工业核心区引发的战略雪球效应，并将“地形”从单纯的阻挡物升级为具有资源与攻防乘数的复合体。

## 2. 核心架构与挑战分析

### 2.1 深度地形与大城市基建 (Cities & Advanced Terrain)
在原本的地形（六角格是草地/森林）基础上，我们需要挂载新的“城市地块（City Node）和行省基建”概念。
*   **深耕的地形修正率**：地形不再只影响移动力。给底层 `RuleEngine` 引入修正系数：
    *   **平原**：装甲部队攻击力 +20%。
    *   **森林/城市**：步兵防御力 +30%，装甲部队强制降速。
*   **城市的实体化**：城市格子拥有 `CityLevel`（大都会、城镇、村落）。
    *   城市拥有了工业产值 (IC, Industrial Capacity) 属性与人力汇聚点 (Manpower Hub) 概念。
    *   争夺鲁尔区、巴黎这样的高等级工业区块，将直接决定交战双方后续回合的输血能力。

### 2.2 初级战时经济系统 (The Wartime Economy System)
为了保持操作爽快感，我们在 0.8 **坚决避免像《维多利亚3》那样去做全套商品物价网**，而是聚焦于最直接的**军工复合体**指标：
*   **引入三大核心国家级资源 (Global Faction Resources)**：
    1.  **人力 (Manpower)**：招募所有步兵/装甲师的基础。随着占领城市每回合缓慢自然增长，伤亡巨大则 manpower 枯竭，部队“有坦克但没人开”，前线无法补员。
    2.  **工业产能 (IC / Industry)**：由占领的工厂或城市提供。这是“产能货币”：用于生产装甲、火炮、汽车，或者修复被炸毁的基础设施。
    3.  **战时补给 (Supplies)**：接轨我们 0.6 建立的后勤线。前线的几十万大军每回合都要吃喝拉撒（消耗 Supply）。如果工业/后方生产不出足够的 Supply，全属性就会崩盘。

### 2.3 生产与补员队列 (Production Queue & Reinforcement)
*   **招募新军 (Unit Deployment)**：
    *   在 UI 增加一个“全国工业部”概览界面。
    *   玩家/元帅 选择花费 `100 IC` 和 `2000 Manpower` 新建一个王牌装甲师。
    *   订单压入生产队列 (Production Queue)，**需要等待 N 个回合 (Turns-to-Build)**。
    *   完成时，该部队仅允许在首都或者拥有高级城市基建的本方大后方 Hex 刷出（强制新部队必须经历往前线长途跋涉的调兵期，给防守方反应时间）。
*   **战损补员 (Reinforcement Loop)**：
    *   前沿师团血量 (Strength) 只有 30% 落荒而逃，只要它撤回己方补给连通的阵地且处于非交战状态，每回合系统会自动从国家的**人力池 (Manpower)** 和**产能池 (IC)** 中扣除一定额度，将其血量慢慢涨回到 100%。

## 3. v0.8 落地细分步调建议 (Sub-version Breakdown)

这是一个庞大的系统工程，建议拆分为以下五步进行小步迭代：

### ✅ **v0.81 战术地理学：地形攻防乘数的深度重构**
先不碰钱和资源，先把底层 `RuleEngine` 战斗计算公式加上地形乘数字典。建立“坦克进山变废铁，步兵蹲城变堡垒”的基础伤害算法，并把这套数据体现在我们在 0.4 打磨的选中视图 UI 面板上。

### ✅ **v0.82 城市的附魔：加入工业产能与人力上限**
扩展 `MapRegion` 与 `HexData` 模型，植入 `industrialValue` 和 `manpowerGrowth` 字段。写一段国家资源统计服务，在每回合结束时 `TurnManager` 自动遍历己方所控制的所有带产值的地块，汇入到 `FactionState (国家总账本)`。

### ✅ **v0.83 吞金巨兽：维持费与前线大补员**
让前线血量不满且非战状态（挂持 `regroup` 命令或在预备队）的单位能自动喝血。喝血的代价就是扣除国家账本的资源。测试发现资源枯竭时，全军补员停止，体现出被强行打出换血劣势的战略困境。

### ✅ **v0.84 大后方兵工厂：队列投产系统的实装**
这需要 UI 和后端的同步支持。完成工厂生产菜单。让玩家可以将空余产能按比例分配在：“生产步兵”、“生产装甲”、“积累战区补给箱”。让新建的部队能够排队 3 回合后空降在柏林或巴黎。

### ✅ **v0.85 工业大厦的灵魂：LLM 内阁与军备部**
我们已经有了 0.5 的元帅大模型，现在是时候加入**内阁部长 Agent (Minister Agent)** 了。把我们国家还剩下多少人、多少厂子，丢给这位大模型经济学 AI。让 AI（比如斯佩尔担任生产部长时）基于它的性格和当前战局吃紧程度，自行计算和建议下个回合是狂造廉价步兵填线，还是憋 5 个回合造一支钢铁洪流！
