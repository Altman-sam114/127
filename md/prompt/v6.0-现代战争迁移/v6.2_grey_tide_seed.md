# v6.2 灰潮行动 2030 默认剧本种子

> 本记录对应 v6.2 的第一批实现。目标是让默认新局进入现代虚构战役数据链，而不是一次性完成发布级 100-220 hex 地图、现代 unit schema、ISR/EW/FireMission 或完整 UI 美术。

## 1. 本轮目标

- 新增 `grey_tide_2030` 现代剧本和 region 数据。
- 让 `DataLoader.loadInitialGameState()` 默认优先加载现代剧本。
- 保留旧阿登 JSON 作为 fallback / 历史兼容资源。
- 让 MapEditor 默认读写现代剧本文件，避免导出后退回旧 `allies/germany` 语义。
- 保持 `Command` / `ZoneDirective -> WarCommandExecutor -> RuleEngine` 权威链不变。

## 2. 已落地

### 2.1 现代默认数据

新增文件：

- `WWIIHexV0/Data/grey_tide_2030_scenario.json`
- `WWIIHexV0/Data/grey_tide_2030_regions.json`

当前种子规模：

- 60 个 hex。
- 10 个 region / operational zone。
- 主要作战方：`blueForce`、`redForce`、`neutral`。
- 关键点：East Airport、Harbor Terminal、Radar Ridge、River Bridge、Industrial Hub、Comms Center、Northern Pass。
- 蓝/红各 5 个初始任务编组。

当前种子仍复用旧 `unit_templates.json` 和旧 `ComponentType`。单位显示名已使用现代任务编组，但底层 template id 仍是兼容字段。

### 2.2 默认加载顺序

`DataLoader.loadInitialGameState()` 当前顺序：

```text
grey_tide_2030_scenario + grey_tide_2030_regions
  -> ardennes_v0_scenario + ardennes_v02_regions
  -> GameState.initial() + 旧 region 叠加 fallback
```

### 2.3 Xcode 资源引用

`WWIIHexV0.xcodeproj/project.pbxproj` 已把两份新 JSON 加入 iOS / macOS app resource phase，避免 release / bundle 路径缺资源。DEBUG 下仍优先从源码 `WWIIHexV0/Data/*.json` 读取。

### 2.4 MapEditor 默认资源桥

- `MapEditorGameResourceBridge` 默认读写 `grey_tide_2030_scenario` / `grey_tide_2030_regions`。
- 现代补给阵营通过 `Faction.dataValue(_:)` 解析。
- `MapEditorExporter` 默认导出 `blueForce` / `redForce` / `neutral`，并把文档中实际出现的阵营补入 `factions`。
- 新建单位默认使用 blue alignment，自动 id 前缀基于 `OperationalSideAlignment`。

## 3. 未完成

- 当前地图只有 60 hex，不满足最终首发建议的 100-220 hex。
- `modern_unit_templates.json`、`modern_commanders.json`、`modern_terrain_rules.json` 还未接入。
- `ComponentType` 仍是 `tank / motorizedInfantry / infantry / artillery`。
- `GamePhase` raw value 仍是 `germanAI/alliedPlayer/resolution`。
- 旧阿登资源仍保留在仓库和 fallback 路径。
- ISR / ContactTrack / EW / FireMission / air tasking / modern logistics 还未实现。

## 4. 验收边界

本轮只能证明：

- 默认数据入口已从阿登切到 `grey_tide_2030`。
- 新现代 JSON 可被静态 JSON 检查解析。
- 新资源已进入 Xcode resource phase。
- MapEditor 默认读写路径已切到现代剧本。

本轮不能证明：

- 灰潮行动已达到发布级地图规模。
- 现代部队、火力、电子战、无人系统或 ISR 已有规则闭环。
- UI 视觉已完成发布级现代 C2 风格。
- 本机 Xcode / XCTest / 模拟器测试已运行。
