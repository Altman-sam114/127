**现状**：`UI/RootGameView.swift` 的 `sidePanel`/`compactPanel` 把 UnitInspector/RegionInspector/CommandPanel/EventLog/AgentPanel 全展开，占屏大。

**做**：
- 新增 `UI/InfoPanelToggle.swift`：`@State var isExpanded: Bool = false` + `[ INFO ]` 按钮。
- 收起时只显 HUD（回合/阵营）+ End Turn 按钮。
- 展开时显现有面板（UnitInspector/RegionInspector/CommandPanel/EventLog/AgentPanel），紧凑布局（ScrollView + 小 padding）。
- 横屏（compactLandscapeLayout）面板区用 InfoPanelToggle 包裹。
- 竖屏（compactLayout）面板 tab 保持，但加 collapse 选项。

**不做**：不改面板内部内容（UnitInspectorView 等不动），只做外层 toggle 容器。

**测试**：手动验证 toggle 正常 + 135 tests 不退步（UI 状态无单测，靠 build + 手动）。
